documentation

Ergatis is a highly scalable and portable architecture for performing high throughput genome annotation and comparative analysis. The architecture supports a large number of commonly used bioinformatics tools and predefines a number of integrated, multi-step pipelines for performing automated gene annotations and comparative analysis. The architecture utilizes a Java based workflow engine and integrates with a popular grid management packages, such as Sun Grid Engine and Condor, in addition to supporting pipeline execution on a stand-alone machine. A web based front-end provides an interface for configuring and monitoring pipelines. The software provides detailed execution status, logging information for process auditing, and facilitates error recovery from point of failure.

Most of the documentation for Ergatis is accessible from a 'documentation' tab like this one but within an actual install of the software. This includes component and pipeline creation tutorials, directory structure convention explanations and component documentation. On this SourceForge site you can find the installation guide and a list of some of the components found within Ergatis (on the right).
